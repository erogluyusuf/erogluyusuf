## Reporting a Vulnerability

I take the security of this project seriously. If you find a vulnerability, please use the following process:

1.  **Do not open a public issue.** This helps protect users until a fix is available.
2.  **Report via Email:** Please send a detailed report to **yusuferoglu1957@gmail.com**.
3.  **Details to Include:** Describe the issue, the steps to reproduce it, and any potential impact.
4.  **Response Time:** You can expect a response within **48-72 hours**.

## Our Process

- After a report is received, I will verify the vulnerability.
- A fix will be developed and tested.
- Once the fix is released, a security advisory will be published, and you will be credited for the discovery (if you wish).
